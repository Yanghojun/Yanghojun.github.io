# Git Documentation

- `[argument]`: 선택적 인자(Optional)
- `<argument>`: 필수 인자(Required)
- `[argument1|argument2]`: 선택적 인자 중 하나(One of Optional argument)
- `<argument>` $ \cdots $: 여러개 인자(Multiple arguments)
- `-argument`: 짧은 옵션(Short option)
- `--arguemt`: 긴 옵션(Long option)
